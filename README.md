# catai

A wrapper for OpenAI Assistant, with function calling for cat images! Users can ask for up to 25 cats.

Stack

- React.js, Tailwind CSS
- FastAPI

Frontend Setup

- npm i
- npm run dev

Backend Setup

- python3 -m venv .venv
- source .venv/bin/activate
- pip install requirements.txt
- uvicorn main:app --reload
